A) Description - top

The mega image viewer jQuery plugin allows you to easily replace <div> tags with animated image viewers.

Viewer used for displaying high-resolution object (image - JPG, PNG, GIF). Viewer displays the given display object inside the user-defined viewport area. Viewer allows to control the position and zoom of the object displayed inside the viewport. Viewer controls the sliding and zoom of the displayed object so that the viewport area will be filled completely.

Features:

  • Touch support for mobile devices.
  • Mouse or Touch drag.
  • Mouse wheel to zoom/unzoom.
  • Mouse click or Touch move to center viewport.
  • Possible viewport resize on runtime.
  • Package contains a example for building thumbnail gallery.
  • Package contains a example for building lightbox gallery (ColorBox Gallery).
  • Cross-browser compatible â€“ Animated Button is compatible with IE, CHROME, FIREFOX, OPERA, SAFARI.
  • Very easy installation on your html page.
  • Very smooth animations.
  • Fully scalable.
  • Displays the map palette to quickly change the view using a thumbnails.
  • Placing hotspots on images.
  • Addition popup windows on the hotspot.
  • Viewer functionality:
    1. set start coordinates and scale
    2. set fit method to viewport
    3. set image scale method
    4. navigation bar (enabled / disabled)
    5. navigation bar autohide
    6. set the navigation bar position (acceptable values: TL, T, TR, L, R, BL, B, BR)
    7. set visibility of the buttons in the navigation bar
    8. set animation duration
    9. set drag inertia
    10. set image source from javascript {"contentUrl" : "images/my_image.jpg"} (relative or absolute path)
    11. set image source from html <img src="images/my_image.jpg"/>
    12. dispatch event 'mivChange' with information whether it is possible to move and zoom the display object

B) Customization settings - top

Parameter Default & Accepted Values Description
viewportWidth string
default: '100%'
accepted: string		 Defines width of the area in which image will be displayed inside the outer div (myDiv). Size can be given in pixels, ems, percentages.
viewportHeight string
default: '100%'
accepted: string		 Defines height of the area in which image will be displayed inside the outer div (myDiv). Size can be given in pixels, ems, percentages.
fitToViewportShortSide boolean
default: false
accepted: true, false		 Shorter side of the displayed object will fit the viewport.
contentSizeOver100 boolean
default: false
accepted: true, false		 If the viewport size (width and height) is greater than the size of the displayed object, allow the object scaled over 100% to fit the viewport (zoom is disabled).
startScale number
default: 1
accepted: 0...1		 Defines start scale.
startX number
default: 0
accepted: integer values		 Defines start coordinate x in px, in the display object frame of reference, which will be moved to the center of the viewport, if it is possible.
startY number
default: 0
accepted: integer values		 Defines start coordinate y in px, in the display object frame of reference, which will be moved to the center of the viewport, if it is possible.
animTime number
default: 500
accepted: integer values		 Defines duration in ms of the scale and position animations.
draggInertia number
default: 10
accepted: integer values		 Defines inertia after dragging.
zoomLevel number

accepted: positive numbers		 Set max zoom level.
zoomStep number
default: 0.1

intNavEnable boolean

intNavPos string

C) Plugin methods - top

Method name Example use Description
setPosition $('#myDiv').lhpMegaImgViewer('setPosition', x, y, scale, noAnim) Sets the position and size of the displayed object. The scale parameter is optional - if null, the size remains unchanged.

@param {integer} x Coordinate x in px, in the display object frame of reference, which will be moved to the center of the viewport (if it is possible).
@param {integer} y Coordinate y in px, in the display object frame of reference, which will be moved to the center of the viewport (if it is possible).
@param {number} scale The size to which the display object will be scaled (if it is possible); optional.
@param {boolean} noAnim Animations disabled; optional, default false.
@return {Object} Returns jQuery object.
moveUp $('#myDiv').lhpMegaImgViewer('moveUp') Initializes the movement of the display object to the top, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveDown $('#myDiv').lhpMegaImgViewer('moveDown') Initializes the movement of the display object to the bottom, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveLeft $('#myDiv').lhpMegaImgViewer('moveLeft') Initializes the movement of the display object to the left, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveRight $('#myDiv').lhpMegaImgViewer('moveRight') Initializes the movement of the display object to the right, to the boundary of the viewport or untill the moveStop method is called.

@return {Object} Returns jQuery object.
moveStop $('#myDiv').lhpMegaImgViewer('moveStop') Stops the movement of the display object.

@return {Object} Returns jQuery object.
zoom $('#myDiv').lhpMegaImgViewer('zoom') Initializes the zooming of the display object up to 100% or untill the zoomStop method is called.

@return {Object} Returns jQuery object.
unzoom $('#myDiv').lhpMegaImgViewer('unzoom') Initializes the unzooming of the display object up to the viewport's size or untill the zoomStop method is called.

@return {Object} Returns jQuery object.
zoomStop $('#myDiv').lhpMegaImgViewer('zoomStop') Stops the zooming/unzooming of the display object.

@return {Object} Returns jQuery object.
fitToViewport $('#myDiv').lhpMegaImgViewer('fitToViewport') Fits the display obejct's size to the viewport size and moves the object to the center of the viewport.

@return {Object} Returns jQuery object.
fullSize $('#myDiv').lhpMegaImgViewer('fullSize') Sets the initial size of the display object and moves the object to the center of the viewport.

@return {Object} Returns jQuery object.
adaptsToContainer $('#myDiv').lhpMegaImgViewer('adaptsToContainer') Control the correct position and size of the object displayed inside the viewport. IMPORTANT - Method should be called after each the outer div resize.

@return {Object} Returns jQuery object.
getCurrentState $('#myDiv').lhpMegaImgViewer('getCurrentState') Returns object with control values. The object is identical as the object returned by the event 'mivChange', read more - Plugin event.

@return {Object} Returns object with control values.
destroy $('#myDiv').lhpMegaImgViewer('destroy') Destructor. Removes the viewer from the page. Restores the original appearance and functionality of the outer <div> element. Allows to efficiently clean the memory.

@return {Object} Returns jQuery object.

D) Plugin event - top

mivReady

This plugin dispatch event 'mivReady' when all data are loaded, DOM completly prepared and plugin ready to use.

EXAMPLE USE: 

$('#myDiv').bind("mivReady", function(e) {
					
});

mivChange

This plugin dispatch event 'mivChange' after modifying the control value. Control values contains information:

  • current display object scale
  • current display object position
  • scaling in progress
  • whether it is possible to move the display object up, down, left or right
  • whether it is possible to zoom and unzoom
  • proportion viewport to display object

This event has the following properties:
scale - number, current scale display object (0...1).
xPosInCenter - number, current x position display object in viewport center (px).
yPosInCenter - number, current y position display object in viewport center (px).
allowUp - true, if the movement in the "up" direction is possible, false otherwise.
allowDown - true, if the movement in the "down" direction is possible, false otherwise.
allowLeft - true, if the movement in the "left" direction is possible, false otherwise.
allowRight - true, if the movement in the "right" direction is possible, false otherwise.
allowZoom - true, if zooming is possible, false otherwise.
allowUnzoom - true, if unzooming is possible, false otherwise.
hPropViewpContent - number, proportion (vertical) viewport to display object.
wPropViewpContent - number, proportion (horizontal) viewport to display object.
isScaled - true, when scaling in progress, false otherwise.

EXAMPLE USE: 

$('#myDiv').bind("mivChange", function(e) {
	alert(e.scale + ', ' + e.xPosInCenter + ', ' + e.allowLeft + ', ' + e.allowRight);
	alert(e.allowDown + ', ' + e.allowUp + ', ' + e.yPosInCenter);
	alert(e.allowZoom + ', ' + e.allowUnzoom);
});

    G) Using hotspots - top

    • STEP 1 - create empty div, set the id and set display none
      This is hotspots container. 
      <div id="MyHotspots" style="display:none;"></div>
    • STEP 2 - create hotspot div, set the class lhp_miv_hotspot and put this div inside hotspots container.
      This is a single hotspot container. 
      <div id="MyHotspots" style="display:none;">
	<div class="lhp_miv_hotspot"></div>
</div>
    • STEP 3 - set position of the hotspot: data-x, data-y
      and set scale above which hotspot is shown: data-visible-scale, (accepted: 0...1).
      <div id="MyHotspots" style="display:none;">
	<div class="lhp_miv_hotspot" data-x="720" data-y="500" data-visible-scale="0"></div>
</div>
    • STEP 4 - add a marker div, set the class lhp_miv_marker
      and specify marker location relative to the hotspot location (accepted: TL, T, TR, L, C, R, BL, B, BR).
      This is a marker container. 
      <div id="MyHotspots" style="display:none;">
	<div class="lhp_miv_hotspot" data-x="720" data-y="500" data-visible-scale="0">
		<div class="lhp_miv_marker pos-BL">MARKER #1</div>
	</div>
</div>
    • STEP 5 (optional) - add a popup div, set the class lhp_miv_popup
      and specify popup location relative to the marker location (accepted: TL, T, TR, L, R, BL, B, BR).
      This is a popup container. 
      <div id="MyHotspots" style="display:none;">
	<div class="lhp_miv_hotspot" data-x="720" data-y="500" data-visible-scale="0">
		<div class="lhp_miv_marker pos-BL">MARKER #1</div>
		<div class="lhp_miv_popup pos-L">POPUP CONTENT</div>
	</div>
</div>
    • STEP 6 add another hotspot.
      Repeat steps 2-5.

    • STEP 7 activate the plugin with hotspots.
      $("#myDiv").lhpMegaImgViewer(settings, "MyHotspots");

    J) Navigation bar - theme - top

    The mega image viewer jQuery plugin uses Themeroller to determine appearance navigation bar.

    If you want change theme:

    1. go to http://jqueryui.com/themeroller/
    2. set your settings and click 'Download theme', you'll be directed to the Download Builder and your custom theme will be auto-selected in the Theme dropdown menu
    3. click 'Download' button and save zip file
    4. unpack zip file and copy the contents of css folder to your css folder
    5. change this link in your head
      <link rel="stylesheet" type="text/css" href="css/custom-theme/jquery-ui-1.10.3.custom.css" />

      <link rel="stylesheet" type="text/css" href="css/redmond/jquery-ui-1.10.3.custom.css" />
